WEB COLOR STORM
可以选择十六进制、RGB和HSL色彩的页面
带有自定义播放器：
实现播放暂停、上一曲/下一曲、点击进度条跳转播放进度功能

/*------------------------------------------------------*/
                        文档说明
/*------------------------------------------------------*/
audio 文件夹下存放了音乐播放器的音频文件（song）和海报图片文件（poster）song
js    文件夹下存放播放器的逻辑代码（audio.js）音频数据文件（audio_data.js）
css   文件夹下audio.css是播放器的样式文件
